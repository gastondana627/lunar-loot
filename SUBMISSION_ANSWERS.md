## Adobe Image Resizing (3:2 Ratio)

### Method 1: Photoshop
1. Open image
2. Image → Image Size
3. Set to: **1800 x 1200 pixels** (or 1200 x 800)
4. Check "Constrain Proportions"
5. Resampling: "Bicubic Sharper (reduction)"
6. Save as JPG (quality 85%)

### Method 2: Photoshop Express (Free)
1. Open image
2. Crop tool
3. Select "Custom" ratio
4. Enter 3:2
5. Adjust crop area
6. Export → JPG

### Method 3: Adobe Express (Online - Free)
1. Go to express.adobe.com
2. Upload image
3. Resize → Custom
4. Enter 1800 x 1200
5. Download

### Quick 3:2 Dimensions:
- 1800 x 1200 (recommended)
- 1500 x 1000
- 1200 x 800
- 900 x 600
