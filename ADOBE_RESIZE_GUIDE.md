## Method 1: Adobe Photoshop (Desktop)

### Steps:
1. **Open your screenshot**
   - File ‚Üí Open ‚Üí Select your screenshot

2. **Check current size**
   - Image ‚Üí Image Size
   - Note the current dimensions

3. **Resize to 3:2 ratio**
   - Width: `1800` pixels
   - Height: `1200` pixels
   - ‚úÖ Check "Constrain Proportions" (lock icon)
   - Resampling: Select "Bicubic Sharper (reduction)"

4. **Save optimized**

   - Quality: 80-90%

## Method 4: Crop First, Then Resize (Best Quality)

If your screenshot isn't exactly 3:2 ratio:

### In Photoshop:
1. **Crop to 3:2 first**
   - Select Crop Tool (C)
   - In top toolbar, set ratio: `3:2`
   - Adjust crop area
   - Press Enter

2. **Then resize**
   - Image ‚Üí Image Size
   - Width: `1800`, Height: `1200`
   - Bicubic Sharper
   - Save

## üö® Common Issues & Fixes

### Issue: "My image is 1920x1080 (16:9)"
**Fix:** Crop to 1920x1280 first, then resize to 1800x1200

### Issue: "File is over 5 MB"
**Fix:** 
- Reduce JPG quality to 70-80%
- Or resize to 1200x800 instead

### Issue: "Image looks blurry after resize"
**Fix:**
- Use "Bicubic Sharper" resampling
- Or apply slight sharpening filter after resize

### Issue: "I don't have Photoshop"
**Fix:** Use free alternatives:
- Adobe Express (online, free)
- Photoshop Express (mobile, free)
- GIMP (desktop, free)
- Online tools: iloveimg.com, resizeimage.net
