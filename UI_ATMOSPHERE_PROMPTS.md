## Generation Tips

### For Freepik/Dreamina/FAL:

1. **Consistency:** Use similar keywords across all prompts
   - "Professional space aesthetic"
   - "Blue-purple lighting"
   - "4K quality"
   - "Cinematic composition"

2. **Atmosphere over Detail:**
   - Focus on mood and lighting
   - Avoid busy/distracting elements
   - Keep backgrounds subtle for text overlay

3. **Color Harmony:**
   - Stick to blue-purple-cyan palette
   - Avoid bright reds/yellows (except accents)
   - Maintain dark base for readability

4. **Technical Specs:**
   - Always request 4K quality
   - Specify 1920x1080px dimensions
   - Request suitable for UI overlay

## Integration Notes

### After Generation:
1. Save all images to `ui_assets/backgrounds/`
2. Optimize for web (compress if needed)
3. Test with text overlay for readability
4. Adjust opacity if backgrounds too bright
5. Ensure consistent mood across all screens

### Text Overlay Guidelines:
- Use white text with dark shadow
- Add semi-transparent dark overlay if needed (rgba(0,0,0,0.4))
- Ensure minimum contrast ratio 4.5:1
- Test readability on all backgrounds
